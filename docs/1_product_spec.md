# Product Specification

## Goal
Build a SaaS platform where meal prep businesses can:
- Launch their own website
- Manage meals and orders
- Offer subscriptions

## Core Features

### Business
- Signup/login
- Create business
- Unique subdomain

### Meals
- CRUD meals
- Include macros

### Menu
- Weekly meal assignment

### Orders
- Place orders
- Track status

### Subscriptions
- Meal plans
- Auto-renew
- Pause/Resume

### Website
- Dynamic UI per business
- Theme + template system
